# Chatbot RAG with Claude Code

Ứng dụng Chatbot RAG dùng Streamlit + OpenAI GPT-4o-mini + ChromaDB.  
Chạy ứng dụng: `streamlit run app.py`

## Rules chi tiết

Xem thư mục `.claude/rules/` để biết đầy đủ:

| File | Nội dung |
|------|----------|
| [project-overview.md](.claude/rules/project-overview.md) | Mục đích, stack công nghệ |
| [architecture.md](.claude/rules/architecture.md) | Cấu trúc thư mục, luồng dữ liệu |
| [environment-setup.md](.claude/rules/environment-setup.md) | Cài đặt, biến môi trường, bảo mật |
| [multi-strategy-pattern.md](.claude/rules/multi-strategy-pattern.md) | Các chiến lược chunking/retrieval/prompt |
| [extending-strategies.md](.claude/rules/extending-strategies.md) | Cách thêm chiến lược mới |
